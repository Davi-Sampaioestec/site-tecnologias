CSS (Cascading Style Sheets) é a linguagem usada para definir o estilo e o layout de páginas web. Com o CSS, é possível controlar aspectos visuais como cores, fontes, tamanhos, espaçamentos, posicionamento de elementos e responsividade. Ele trabalha junto com o HTML: enquanto o HTML estrutura o conteúdo, o CSS cuida da aparência. O CSS pode ser aplicado diretamente nas tags HTML, dentro do próprio arquivo HTML ou em arquivos externos.

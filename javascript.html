Tipografia

JavaScript é uma linguagem de programação usada para tornar páginas web interativas e dinâmicas. Diferente do HTML (estrutura) e do CSS (estilo), o JavaScript permite adicionar funcionalidades como animações, validação de formulários, menus interativos, atualizações em tempo real e muito mais. Ele é executado no navegador do usuário (client-side), mas também pode ser usado no servidor (com Node.js, por exemplo). É uma das principais tecnologias da web, junto com HTML e CSS.

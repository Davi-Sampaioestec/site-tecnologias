Tipografia

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas e conteúdos na web. Ele serve para estruturar as informações de um site, como textos, imagens, links, tabelas, formulários e outros elementos. O HTML usa "tags" para marcar e organizar os diferentes componentes da página, permitindo que os navegadores saibam como exibir o conteúdo para os usuários.
